(static) onSessionOpened()

Called when the web client's session has opened. This in turn calls the listener provided by the web client when opening a session. The web client needs to open a session after it is successfully bound with Dynamix. The client can then create a handler using which it can make context requests to the Dynamix Framework. NOTE : This will be automatically called after the security authorization is granted by the user (if required).
